Error - CSS file missing

You are seeing this screen because the launcher is unable to load the .css file properly. Here's some of the possible causes:

  • The file has been moved.
  • The file has been deleted.
  • The file has been corrupted.

The launcher will not function properly without the following files:

  • Scripts/DL_Stylesheet.css
  • Scripts/DL_Stylesheet_Animations.css
  • Scripts/DL_Stylesheet_Reference.css
  • Scripts/DL_Javascript.js

Please download a fresh copy of the launcher.
