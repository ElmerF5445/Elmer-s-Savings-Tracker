How data is stored

Doodle Launcher stores the following data:

  • Category title text and shortcut title text.
  • URLs associated with the shortcut created.
  • Settings values for both appearance and behaviors.
  • Launcher version data, build number, version title.
  • Session data (whether the launcher is opened for the first time since the browser is opened).

How these data is stored:

  • Doodle Launcher uses your browser's local storage feature. This feature allows a page to store values and texts in your storage.
  • Using different browsers doesn't transfer the data into that browser.
  • Any data stored in the browser local storage is bound within the restrictions of the browser. This prohibits the launcher from accessing any file in your computer.
  • This data can easily be accessed by other pages and websites. It is highly recommended to always check the URL preview before opening a shortcut. It is also recommended to never input sensitive information such as addresses or account passwords into the system.

Is my private information being collected?

  • The launcher does not collect any information besides the information you input.
  • The launcher only use the browser's local storage, and it does not send any information to anyone else.

How to use Shortcut Editor

There are two tabs in Shortcut Editor:

  • Normal view previews how your shortcuts would look like in the Home page.
  • Table view lets you edit or delete items in your shortcut library.

There are two types of items you can create:

  • A category is a toggleable tab where you can group your shortcuts.
  • A shortcut is a button that opens a new tab with the URL you provide when clicked.

How to create categories:

  • On the top of the page there is a button named "Add Item". Click it and a window should appear. It doesn't matter what tab you are in.
  • Inside the window, click "Category".
  • Type the name of your category on the input field, then click "Create Item".
  • You should receive confirmation via a toast notification that the item has been successfully made (unless you have toast notifications disabled in the settings).

How to create shortcuts:

  • On the top of the page there is a button named "Add Item". Click it and a window should appear. It doesn't matter what tab you are in.
  • Inside the window, click "Shortcut".
  • There will be three input fields present. The shortcut title, URL, and category.
  • The shortcut title is the displayed text of the shortcut button.
  • The shortcut URL is the URL the shortcut button links to when clicked.
  • The shortcut category is the category where this new shortcut button is to be inserted. It is required to choose a category (you must create one if you haven't already).
  • The shortcut category can be clicked, and a dropdown menu will appear. Click the category name where you want your shortcut to be inserted. The dropdown will close and the selected category will appear on the button.
  • All three inputs are required. Once done, click "Create Item".
  • You should receive confirmation via a toast notification that the item has been successfully made (unless you have toast notifications disabled in the settings).

How to edit items using Table View:

  • On the side of the screen, click the table icon to switch to Table View.
  • It should now show a table version of your shotcut library. The tables display the shortcut name and its URL, along with the editing controls.
  • The eye icon lets you open the shortcut as if you are clicking the shortcut button in Normal View.
  • The pencil icon lets you edit the data of the item, while the trashcan icon deletes it.
  • Click the pencil icon beside the item you want to modify and it should open a window.
  • Modify the shortcut name or the URL or change the category name if that is what you selected.
  • Once done, click "Save Changes".
  • You should receive confirmation via a toast notification that the item has been successfully made (unless you have toast notifications disabled in the settings).
  • To delete items, simply click the trashcan icon beside the row you want to delete. If a confirmation box appears, click "Delete". You should receive confirmation via a toast notification that the item has been successfully made (unless you have toast notifications disabled in the settings).

What is Doodle Launcher

Doodle Launcher is an HTML based app that enables you to create an organized library of your frequently used links. You can create shortcuts that links to any URL you want. You can organize these shortcuts into toggleable folders.

Doodle Launcher is currently in Beta, but it has been in development since October 2021, although it was less of a dynamic system back then.

Shortcut Library Profiles (0.1)

This feature lets you create different shortcut libraries and seperate them from each other. Like a login account system, you can secure your shortcut library by using a password to lock your shortcut library.

Enabling this feature adds the following:

  • The profile login UI that displays every beginning of the session.
  • A Profile tab on Settings that contains all the tools and settings needed for the feature.
  • Changes the shortcut system to take profiles into account.
  • Changes how your shortcuts and categories are saved to take profiles into account.

This feature is unfinished. Here's some notable problems:

  • Enabling this feature will make it impossible to retrieve your old shortcuts when you disable it again.
  • Any shortcuts made before and while the feature is enabled will be impossible to recover or used even if you disable it.
